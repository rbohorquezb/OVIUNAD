¿Que es Hub?

Repository. Un repositorio o “repo” es un directorio donde se almacenan los archivos de tu proyecto. Puede estar ubicado en el almacenamiento de GitHub o en un repositorio local en tu computadora Si Git es el corazón de GitHub, entonces Hub es su alma. El hub de GitHub es lo que convierte una línea de comandos, como Git, en la red social más grande para desarrolladores.

GitHub

GitHub es la plataforma de desarrollo colaborativo de software en donde usando el sistema de control de versiones de Git podemos gestionar mejor nuestro código además de alojarlo y compartirlo con la comunidad En la actualidad, GitHub es mucho más que un servicio de alojamiento de código. Además de éste, se ofrecen varias herramientas útiles para el trabajo en equipo.

Uso de GitHub

Primero vamos a github.com y creamos un nuevo repositorio para nuestro código, le pondremos un nombre a nuestro repo, se recomienda que el nombre del repositorio sea igual al nombre de proyecto en nuestro ordenador. En nuestra especialidad “Programación”, fuimos aprendiendo cosas y creando programas de código abierto.fomentando el software libre y proyectos para amateur.
